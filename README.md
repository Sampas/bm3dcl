# BM3DCL
OpenCL implementation of BM3D image denoising algorithm.

* Developed mainly on NVIDIA drivers, CUDA installation required.
* Use CMake to generate project files for compiling.

## Usage

```sh
./bm3dcl <noisy> [sigma] [original]
```
